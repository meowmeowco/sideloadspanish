# Seed

## Vision

A Chrome/Zen browser addon that replaces words on web pages from a source language to the target language the user is learning. Replacement difficulty and frequency scale with the user's progress — starting with common, easy words and gradually introducing harder vocabulary as mastery builds.

## Core Concept

- Browse the web normally; the addon transparently swaps words in-page
- Difficulty adapts: beginners see few, high-frequency replacements; advanced users see more and harder ones
- Progress tracking drives the replacement algorithm
- Hover/click on a replaced word to see the original and definition
- Works across any website content

## Arguments

```
a chrome/zen addon that replaces some words from source language to the target language you learn, depending on the difficulty and progress
```
